# assignment_autofiller

Smart Job Autofill Extension
A Chrome extension that identifies job application forms on web pages and automatically fills them with your saved personal details such as Name, Email, Phone, and Skills.

How to Install
1. Open Chrome Extensions
    * Launch Google Chrome.
    * Go to chrome://extensions/ in the address bar.
2. Enable Developer Mode
    * Turn on the toggle labeled Developer mode in the top-right corner.
3. Load the Extension
    * Click Load unpacked in the top-left corner.
    * Select the folder assignment_autofiller that contains manifest.json.
4. Check Installation
    * You should see Smart Job Autofill listed among your installed extensions.
